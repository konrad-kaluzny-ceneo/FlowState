# Branding FlowState

## Esencja marki

FlowState pomaga odzyskać kontrolę nad dniem pracy.

Nie próbuje zwiększać liczby wykonanych zadań.
Pomaga wykonywać właściwe zadania, we właściwym czasie i bez ciągłego przełączania uwagi.

FlowState jest narzędziem dla osób, które są zmęczone chaosem współczesnej pracy.

Nie organizuje wszystkiego.

Pomaga skupić się na jednej rzeczy.

---

# Archetyp marki

Opiekun + Mentor

FlowState nie motywuje.

Nie pogania.

Nie ocenia.

Towarzyszy użytkownikowi podczas pracy i pomaga zachować spokój.

# Opis aplikacji

FlowState to aplikacja do świadomego planowania dnia i pracy w głębokim skupieniu.

Nie próbuje zmusić Cię do robienia większej liczby rzeczy.

Pomaga wykonywać właściwe zadania, we właściwej kolejności i bez ciągłego przełączania uwagi.

Łączy planowanie dnia, sesje skupienia, zdrowe przerwy oraz prosty system zarządzania zadaniami w jedno spokojne środowisko pracy.

Dzięki temu kończysz dzień z poczuciem kontroli zamiast przeciążenia.

# Ton komunikacji

FlowState nigdy nie mówi:

❌ Zwiększ produktywność.

❌ Pracuj szybciej.

❌ Osiągaj więcej.

❌ Wykonuj więcej zadań.

Zamiast tego mówi:

✓ Wybierz jedną rzecz.

✓ Oddychaj.

✓ Zacznij spokojnie.

✓ Skup się.

✓ Reszta może poczekać.

✓ Pracuj we własnym rytmie.

✓ Dzisiaj wystarczy jedno naprawdę ważne zadanie.
